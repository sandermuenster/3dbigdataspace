# Data sources

| Module | Provider | Access | Notes |
|---|---|---|---|
| `europeana` | Europeana | REST Search API | needs `EUROPEANA_API_KEY` |
| `smithsonian` | Smithsonian Open Access 3D | REST | GLB assets |
| `oai_pmh` | any OAI-PMH endpoint | OAI-PMH (Dublin Core) | presets incl. nfdi4ing |
| `scan_the_world` | MyMiniFactory | REST | needs `MYMINIFACTORY_TOKEN` |
| `hunt_museum` | Hunt Museum | HTML scrape | best-effort selectors |
| `google_landmarks` | Google Landmarks v2 | CSV index | URLs only, no bulk image download |
